# Tripleten web_project_homeland

Descripción
Este proyecto es una página web responsiva llamada “Homeland”. Muestra una galería de lugares de diferentes partes del mundo e introduce historias de personas de distintas regiones.

Tecnologías utilizadas
HTML5
CSS3
Metodología BEM
Diseño responsivo (Flexbox + Media Queries)
Enlace del proyecto

https://github.com/Kelvin-Arias-Z/web_project_homeland

Cómo ejecutarlo localmente
Clona el repositorio:
git clone https://github.com/Kelvin-Arias-Z/web_project_homeland
Abre la carpeta del proyecto
Abre index.html en tu navegador
